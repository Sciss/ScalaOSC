h3. requirements / installation

ScalaOSC currently cross-builds against Scala 2.9.1, 2.9.0 and 2.8.1. The repository contains a project file for Intellij IDEA 10 CE, and project files for sbt 0.10 (xsbt). Standard targets for sbt:

* @clean@
* @compile@
* @doc@
* @package@
* @run@ __arguments...__
